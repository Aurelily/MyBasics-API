Cet API est une API en PHP Natif avec un routeur fait à la main à base de paramètre GET et d'une page traitement.php
Exemple : pour récupérer tous les users avec la fonction fetchAllUsers du front en ReactJS ou quel qu'il soit, on doit appeler :
http://localhost/MyBasics-API/myBasic-php-mysql/traitement.php?users  et non pas une route classique comme avec NodeJS
